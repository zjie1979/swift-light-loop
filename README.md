# 无限循环饮食法——迅猛轻盈版

一个纯静态、可离线使用的三日循环饮食计划 PWA。打卡、轮次和进食窗口只保存在浏览器本地，不需要账号或服务器。

## 核心功能

- Day 1 / Day 2 / Day 3 循环食谱
- 每日两顿正餐 + 可选极简早餐
- 可调整的 16+8 进食窗口
- 本地打卡、轮次统计和完整一轮反馈
- iPhone Safari 添加到主屏幕
- 健康风险提示和权威资料链接

## 隐私

本项目不收集、不上传任何个人记录，也不包含分析脚本、广告或第三方登录。

## 重要说明

本应用只整理用户提供的饮食计划，不构成医疗或营养建议。极低热量饮食和间歇性进食并不适合所有人，请阅读应用内安全提示，必要时咨询医生或注册营养师。
